# 上海中心活动

## 体验地址

![小程序太阳码](/shanghaitower/qrcode.jpg)

## 作品介绍

这个小程序为上海中心提供了两场互动活动，垂直马拉松&共创大会。用户可以在小程序上传脸部档案然后通过 AI 制图，最后由工作人员帮助打印照片给用户。
